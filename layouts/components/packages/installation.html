Installation

To install this package, start R (version "<%= r_ver_for_bioc_ver(@package[:bioc_version_num]) %>") and enter:

if (!require("BiocManager", quietly = TRUE)) install.packages("BiocManager") <% if (@package[:bioc_version_str] == "Development") %> # The following initializes usage of Bioc devel BiocManager::install(version='devel') <% end %> BiocManager::install("<%=@package[:Package]%>")

For older versions of R, please refer to the appropriate Bioconductor release.
