Installing software

For notes on software installation, follow the "HowTo" link on the Bioconductor website. The following software should be installed for the Bressanone course.

  • From CRAN: R version 1.9.0 (Linux, Windows, MacOS).
  • From Bioconductor
    • All Bioconductor packages from release 1.4, using the install script: getBioc(libName="all").
    • Annotation data packages: hgu95av2, hgu133a, GO.
    • Experimental data packages: ALL, estrogen, lymphoma.
