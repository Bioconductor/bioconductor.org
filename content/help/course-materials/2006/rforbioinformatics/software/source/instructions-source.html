Downloading Files

  • The sources for R-devel: R-2.3.r36984.tgz

Building from source

Unpack the R sources as follows:

tar xzvf R-2.3.r36984.tgz

Create a directory for the build:

mkdir R-2.3

Run configure from the R-2.3 directory. If you see any errors, it may indicate that you are missing some required system headers and/or libraries. After configure runs without error, you should run make.:

cd R-2.3
../R-2.3.r36984/configure
## assuming no errors
make

Testing your R build

Try starting R:

R-2.3/bin/R

Special Steps for OSX Tiger 10.4.x

  1. Install Xcode 2.0 or greater. The Xcode tools can be found on your OSX Tiger DVD or it can be downloaded from Apple (you will need to create a free acount).

  2. Select gcc3 as your default compiler:

    sudo gcc_select 3.3

  3. Install X11. You can install the X11User.pkg from the OSX Tiger DVD, or download a copy from Apple here.

  1. Install a fortran compiler. The easiest way to get the proper fortran compiler installed is to install a binary version of R.app from CRAN. You can either select to install it during the installation of R.app or you can install it directly by browsing the Packages directory in the R.app disk image and clicking on the g77 package.
  2. Install LaTeX. We recommend using the i-Installer LaTeX package. First, download i-Installer. Then select the TeX package and install it.
  3. Build R (see instructions above).
