Software for the R for Bioinformatics Course

This page describes how to install the software that will be used during the R for Bioinformatics Course.

The objective is to install the tools needed to develop R software packages. This is more complicated than just installing R, becasue as a developer, you need to be able to build packages and this requires an additional set of software tools.
