The Bioconductor Project uses the R statistical programming language. Bioconductor releases are therefore closely tied to a particular version of R. Packages contributed to CRAN can be used as dependencies in Bioconductor packages and vice versa.

The R Project for Statistical Computing

Bioconductor and Galaxy are meeting quartly to try to increase the collaboration between the two projects. Bioconductor workshop infrastructure is modeled after the Galaxy infrastructure.

Galaxy

Bioconductor is available for use within the AnVIL project. Bioconductor/R is directly available through Bioconductor docker images available on Terra. Bioconductor makes use of binary package generation to allow quick installation of any Bioconductor package and has provided numerous workflows for data analysis.

AnVIL
